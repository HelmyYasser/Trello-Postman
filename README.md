## 📌 Trello API Postman Collection

This project contains a **Postman collection** for automating and testing the Trello REST API workflow.

It covers full CRUD operations for:

- 🗂️ Boards  
- 📋 Lists  
- 📝 Cards  
- ✅ Checklists  

The collection uses **environment variables** to dynamically store and reuse IDs between requests, simulating a real-world API testing flow.

Each request includes automated tests to validate:

- ✔️ Status code (200)  
- ⚡ Response time performance  

### 🚀 Key Skills Demonstrated

- REST API testing  
- Postman automation  
- Environment variable handling  
- Basic performance validation  

This project is suitable for API testing practice, QA automation portfolios, and CI/CD integration using Newman.
